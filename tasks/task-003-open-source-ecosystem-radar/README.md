# Open Source Ecosystem Radar

**Benchmark question:** Which open-source repositories or ecosystems show unusual growth, decline, concentration risk, or emerging momentum in the selected time window?

This task includes sample and full-mode scripts, an answer schema, and a rubric. Raw data is not committed.

Required sections:

- Executive Summary
- Dataset Scope
- Ecosystem Trends
- Repositories Heating Up
- Repositories Cooling Down
- Maintainer or Contributor Concentration Risk
- Event Timeline
- Evidence-backed Claims
- Caveats and Data Limitations
- Context Engineering Strategy
