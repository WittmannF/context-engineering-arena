# Submission Guide

Submit under `submissions/<participant-id>/<task-id>/`. Required: `answer.json`, `context_trace.json`, `strategy.md`. Optional: runnable code, requirements, notebooks, artifacts, logs. Good claims cite concrete evidence and label confidence. Bad claims assert intent, use vague citations, or hide uncertainty. Validate with `python -m arena_cli.cli validate-submission --participant <id> --task <task-id>`.
