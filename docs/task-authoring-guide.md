# Task Authoring Guide

A good arena task has a large or complex corpus, messy heterogeneous data, a high-level synthesis question, evidence-backed claims, objective evaluation structure, a visual final page, meaningful uncertainty, and public or safely redistributable data. Avoid trivia, one-search questions, private data, unsafe conclusions, and unverifiable evidence. Define schemas, sample mode, licensing, ethics notes, scoring metrics, and a reproducible data pipeline. Use `tasks/proposals/task-template/` as a starting point.
